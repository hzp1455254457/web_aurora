## 🔍 检查清单

在浏览器中查看页面时，请检查以下内容：

### 布局检查
- [ ] 页面整体布局是否居中
- [ ] 头部高度是否为 84px
- [ ] Logo 是否正确显示
- [ ] 主内容区域宽度是否为 1344px（在 1440px 屏幕上）

### 样式检查
- [ ] 头部是否有阴影效果
- [ ] 背景色是否为白色
- [ ] 各区域间距是否正确（48px, 66px, 102px）
- [ ] 内容区域是否按设计排列

### 响应式检查
- [ ] 缩小浏览器窗口，布局是否正常适配
- [ ] 移动端视图是否正常显示

1. **并排对比**：在浏览器中打开页面，同时在 Figma 中打开设计，并排对比
2. **使用开发者工具**：按 F12 打开开发者工具，检查元素尺寸
3. **检查响应式**：调整浏览器窗口大小，测试不同屏幕尺寸

## 🐛 如果发现问题

如果发现页面与 Figma 设计不符，请检查：

1. **浏览器缓存**：按 Ctrl+F5 强制刷新
2. **控制台错误**：查看浏览器控制台是否有错误
3. **样式覆盖**：检查是否有其他样式覆盖了设计样式
4. **图片加载**：确认 Logo 图片是否正确加载
